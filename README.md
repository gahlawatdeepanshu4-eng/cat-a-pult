# Cat-a-pult

Drag the catapult back, let go, land the cat on the target. Five levels.
A single-player landscape game for Android, installable to the home screen.

## Play locally

```bash
npm run serve
```

Then open http://localhost:5173 in a landscape phone-sized window.

To play on your phone, find your PC's local IP and open `http://<that-ip>:5173`
on the same wifi.

## Test

```bash
npm test
```

No dependencies. Tests run on Node's built-in runner.

## Regenerate the icons

```bash
npm run icons
```

The icons are generated by a raw PNG encoder built on Node's `zlib`, so there
is no binary asset that cannot be rebuilt from source.

## Rules

Pull back and let go, like a slingshot. Drag direction aims, drag length sets
power, and there is no timer.

Cat 20 points, T-rex 50. Clear every creature before the rocks run out.
Running out just restarts the level: no lives, no game over.

Twenty levels. Dodging ramps up from level 1, jumping starts at level 2, and
from level 4 there are two flying cats and two flying T-rexes every level.
